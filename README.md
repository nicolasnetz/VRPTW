# VRPTW

Solve the Solomon instances with the model VRPTW1 from Ch.5 Toth&Vigo, 2014 using python and cplex

Features:

* csv outputs in my_result.csv
* plot outputs
* model outputs
* solution outputs (route and vehicle usage)
